Ride On Time
===

Parsing GTFS for great justice!

Installation
---

* Clone from master
* `cd` into the directory and `pip install -r requirements.txt`
* Export a secret key, like so `export RIDEONTIME_SECRET_KEY='when is the bus getting here?'`
* 'cd' into 'project' directory
* `./manage.py check`
* `./manage.py test`
